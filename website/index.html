This website is served from my HTTP server

It is a "loosely" compliant HTTP/1.0 server. I used this website as my main source on the details of the standard.

But there are a lot of things that I'm not implementing to spec. For example, HTTP 1.0 compliant servers should also understand HTTP 0.9 style requests. And my server doesn't. I'm also not covering all headers and such. So it's a pretty minimally working HTTP server.
